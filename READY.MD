# Diseño y Desarrollo del Portafolio Digital / Landing Page

Este documento detalla el plan para crear la plataforma web corporativa para **Jairo Galindez Mahecha - Ingeniería Electromecánica**. El objetivo es construir una Landing Page de alto impacto, enfocada en el sector industrial, que sea moderna, rápida y adaptable a dispositivos móviles.

## Enfoque de Diseño (Aesthetics)
- **Tema Visual:** Diseño "Industrial Premium". Utilizaremos una paleta de colores oscuros (fondos gris oscuro/negro) con acentos vibrantes (como amarillo industrial o azul eléctrico) para transmitir energía, profesionalismo y confianza.
- **Tecnología:** HTML5, CSS3 (Vanilla puro para control total y alto rendimiento), y JavaScript para interactividad.
- **Interactividad:** Micro-animaciones suaves al desplazarse por la página, efectos de hover en las tarjetas de servicios y una interfaz de usuario dinámica (Glassmorphism para barras de navegación, etc.).

## Estructura de la Landing Page

1. **Header (Navegación):** Menú pegajoso (sticky) con enlaces a cada sección y un botón de "Cotizar ahora" (llamado a la acción de WhatsApp).
2. **Sección Hero (Inicio):** 
   - Título de alto impacto: "Soluciones Integrales en Ingeniería Electromecánica".
   - Subtítulo destacando a Jairo Galindez Mahecha.
   - Espacio reservado para una fotografía profesional en campo.
3. **Sobre Mí / Identidad:**
   - Resumen de años de experiencia y diferenciadores clave (Puntualidad, conocimiento, garantías).
   - Mención de certificaciones (RETIE, trabajo en alturas, etc.).
4. **Catálogo de Servicios:** Tarjetas dinámicas con los 3 módulos principales:
   - *Plantas Eléctricas:* Instalación, mantenimiento, automatización.
   - *Energía Solar:* Paneles, mantenimiento, cálculos.
   - *Otros Servicios:* Tableros de control, instalaciones eléctricas, bombas.
5. **Proyectos Destacados (Casos de Éxito):**
   - Espacio para 2-3 casos de estudio clave (con placeholders para fotos de "Antes y Después").
6. **Contacto y Cobertura:**
   - Zonas de cobertura y botones de acceso directo a WhatsApp y Correo Electrónico.
